# Linkly

A URL shortening service that streamlines your online experience.

## Demo

[Linkly Demo](https://abinesh-antino.github.io/web/)


## Features

-   URL shortening
-   Easy-to-use interface
-   Dark mode support
-   Clipboard integration
-   Auto-paste from clipboard
